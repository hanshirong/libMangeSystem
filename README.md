# libMangeSystem
a system to manage own books
- 前端vue
- 后端express
### 后端搭建过程：
1. `npm install express -generator -g`
2.  `express name `
3. cd name
4. `npm install`
5. `npm start`
6. `npm install nodemoe`
7. 在package.json的scripts里面添加 "dev": "nodemon"
8. 启动`npm run dev`
### 前端搭建过程
1. `vue init webpack name`
2. 根据提示填就好啦
3. cd name
4. 启动`npm run dev`
